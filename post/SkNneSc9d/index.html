又建了一个博客站点

2019-04-17

花了点时间，在全球最大的同性交友网站 Github 上，用 Pages 服务建立一个静态博客站点。又花了几十块 RMB 在 Godaddy 上买了域名 gentleming.me，算是搭建了一个小博客网站。

为什么要说又呢？

其实几年我用 Hexo 在 Github 上建过一个博客，当时买了域名 gentleming.com，嗯，当时这个域名没有被人注册，现在只能从域名商购买了，价格不划算，而且 .me 的域名也挺酷的，所以这次就用 gentleming.me了。

上次搭了博客，就搁置在那，没写几篇文章，最后就没有最后了。这次重新建博客是因为发现了 Gridea，这是一个可以在 Mac 或 Windows 上同步 Github Pages 的博客管理、写作工具，使用下来我觉得非常好用 👍，简单方便，设计又好看，于是乎重新点燃了我写博客的激情，ok，那么就希望这次能坚持下去吧。
